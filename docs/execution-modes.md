# Execution Modes

OPOS v1.0 supports both imperative and declarative orchestration targets.

Imperative examples:
- Airflow
- Prefect
- Dagster

Declarative/container-native examples:
- Kestra
- Argo Workflows
- Kubeflow Pipelines
- Flyte

OPOS itself does not require OTS. `ots_export` is optional.
